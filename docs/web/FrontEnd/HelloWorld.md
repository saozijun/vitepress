### 🥳恭喜恭喜🎉
你已经成功的写出你的第一个网页。  

你可能会说：“就这？”  

其实我们还没有使用上【HTML】，只不过创建了一个HTML文件，但这是为了方便让你理解。“哦！它这个网页文件是可以编辑的”

编辑的时候修改后缀名为txt  
展示的时候修改为：html  
这显然很麻烦...  
所以我们需要【代码编辑器】，这样既可以打开来编辑，也可以打开成【网页】来给我们看。
